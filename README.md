# toDo

ToDo App
This is a simple ToDo app built with React.

Overview
The app allows users to:

Add new ToDo items
View existing ToDo items
Delete ToDo items
Each ToDo item displays the text and has a delete icon.

Technologies
React - Frontend framework
Font Awesome - For icons
